Local Movie Sets

Creates Jellyfin collections from local NFO metadata managed by tinyMediaManager. No internet connection needed.
Tip: Disable "Automatically add to collection" in your movie library settings to avoid conflicts with TMDB-based collections.

Path to tinyMediaManager's "Movie Set Data Folder" setting. This folder contains per-set NFO files and artwork. Leave blank if you only use the <set> tag in individual movie NFOs.
Must match the "NFO file naming" option configured in tinyMediaManager → Settings → Movies → Movie Sets.
A collection is only created if at least this many movies in your library belong to the set. Default: 1.
Remove Jellyfin collections that no longer have any movies with a matching <set> tag. Only affects collections without a TMDB or IMDB provider ID (i.e. those created by this plugin).
When enabled, artwork from the TMM set folder is re-copied on every sync, replacing any existing images on the collection. When disabled, only collections with no artwork are updated.
